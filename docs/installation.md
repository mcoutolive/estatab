# Installation

## Stable release

To install estatab, run this command in your terminal:

```sh
uv add estatab
```

Or if you prefer to use `pip`:

```sh
pip install estatab
```

## From source

The source files for estatab can be downloaded from the [Github repo](https://github.com/mcoutolive/estatab).

You can either clone the public repository:

```sh
git clone git://github.com/mcoutolive/estatab
```

Or download the [tarball](https://github.com/mcoutolive/estatab/tarball/master):

```sh
curl -OJL https://github.com/mcoutolive/estatab/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd estatab
uv pip install .
```
